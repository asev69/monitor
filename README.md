# monitor
A simple static HTTP server written in Go

## Build
```
go get -u github.com/asev69/monitor
go build github.com/asev69/monitor
```

## Run
```
./monitor -h
Usage of ./monitor:
  -d string
    	directory to serve (default "/usr/share/doc")
  -p string
    	port to listen (default ":8080")
```
